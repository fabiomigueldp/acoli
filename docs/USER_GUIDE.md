# Guia rapido

## Coordenador
1) Configure comunidades, funcoes, posicoes e perfis de requisito.
2) Crie modelos de missa (recorrencias) e gere instancias futuras.
3) Use o wizard de eventos para triduos/novena/oitava com resolucao de conflitos.
4) Cadastre acolitos, qualificacoes e vincule usuarios aos perfis.
5) Rode o escalonamento em "Escalonar" e publique a escala.
6) Acompanhe confirmacoes e trate cancelamentos/substituicoes.
7) Use a tela "Substituicoes" para resolver vagas abertas rapidamente e marcar resolucoes.
8) Ajuste as configuracoes da paroquia (janelas, pesos, notificacoes) conforme necessario.
9) Consulte a tela "Auditoria" para rastrear mudancas importantes.
10) Rode o job de consolidacao para travar as missas dentro da janela.
11) Use "Escala geral" para ver buracos rapidamente e exportar PDF/WhatsApp.
12) Use "Relatorios" para acompanhar frequencia e confiabilidade.

## Acolito
- Veja suas escalas em "Minhas escalas".
- Confirme, recuse ou cancele quando necessario.
- Solicite troca com outro acolito ou troca de funcao.
- Atualize disponibilidade e preferencias.
- Marque interesse em series de eventos para receber convites.
- Gere o link do calendario para assinar suas escalas.

### Disponibilidade e preferencias
- "Disponivel apenas" restringe sua agenda: voce so sera escalado dentro dessas regras validas.
- "Indisponivel" bloqueia datas/horarios e sempre tem prioridade.
- Ausencias por data sempre bloqueiam, mesmo que exista rotina semanal.
- Horario final nao inclui o horario exato (ex.: ate 10:00 significa antes de 10:00).

### Calendario (ICS)
- Em "Minhas escalas", gere o link do calendario.
- Use o link para assinar no Google Calendar ou Apple Calendar.
